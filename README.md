# kimtyweter.github.io
# 使用模板
這次的網站使用了網路上的免費bootstrap模板，主要的個人頁面使用的是這個網站：https://startbootstrap.com/themes/creative/
，並且做了文字上的修改，以及更改子網頁的目錄。分別是：Introduction, About Me, 相片, 作品集, Contact
而在網頁中也有置入一個網站是存放攝影作品，也是使用了免費的bootstrap的網站：https://colorlib.com/wp/template/capture/
，我把自己的攝影圖片上傳上去，刪減了原本模板上其他的子目錄，只留下Home跟About作為我的自我介紹還有作品集

# 製作過程
將主要頁面模板套入，並且根據自己的需求分成：Introduction, About Me, 相片, 作品集, Contact

Introduction:簡略的描述自己的特點，以簡單的兩句話來吸引讀者目光，並且將背景圖片換成自己的個人照片
About Me：分成學歷、程式語言、語言水準、社團與實習 四個面向去呈現自己
相片：將自己的六張大學生涯照片放入，並且增加了圖文的說明。
作品集：更改了模板的內容，製作成超連結可以讓讀者點擊進入
Contact：聯絡資訊，將模板的樣式更正為自己的

另外，在此說明作品集的製作過程：
1. 精選12張自己的作品
2. 把模板多餘的功能刪去，只留下主頁面呈現、子頁面的點擊畫面
3. 修改主頁面與子頁面的文字敘述、圖片
4. 變更連結與icon連結

# 呈現構思：

在個人網頁的主頁上，我想呈現出活潑開朗的個性，因此我的選色用了「亮橘色」做為代表。橘色帶給人溫暖熱情的感覺，搭配上白色文字，在閱讀方面來看也相當明顯。精選的六張照片也是以能夠表現出自己才華的照片為主，像是體育競賽、出國交換，以及呈現出自己在社團上的經營，如：返鄉服務、蘭陽之夜等。
而在作品集的呈現上，我選擇用粉紅色為主體，粉紅色是我的個人特色，我喜歡粉紅色帶給人的稚氣與包容，多元繽紛的感覺特別適合用在照片的呈現。
