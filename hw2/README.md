#使用模板
本次的作業沒有使用任何模板，而是以老師給的範例為基礎，進行CSS、HTML方向的兩個面向的簡單修改。

#製作過程
將老師的模板先複製到自己的Studio code裡面並且觀察老師的code在實際中有哪些可以修改的。並且在code檔案裡面先註解有可以發揮創意的地方。其次再將老師指定的給同學的API設定按照規定改成自己的學號，作為自己資料庫的輸入、輸出根據。

#設計理念
粉紅色是我這次的主要呈現顏色，之所以想要用粉紅色為底色是因為想要表達出我的購物網站的多元與自由。從第一次世界大戰後，在性別角色裡粉紅色通常與女性氣質相聯繫，正如藍色通常與男性氣質連想在一起。可是時間演變自今，我們不再有這些性別刻板印象的枷鎖，每個人都能夠有喜歡粉紅色的自由。正如同我的網站，沒有死板、保守的經營理念，而是在購物的體驗當中可以感受到愛與包容，乃至於自由！

#呈現構思：
我比照老師的樣板，將點選的區塊、文字的區塊都用方框來呈現。方框的形狀正正方方，在新增與點選之際讓人有一種「穩重」、「踏實」的感覺。
